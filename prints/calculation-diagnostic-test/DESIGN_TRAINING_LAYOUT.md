# 訓練プリント 用紙レイアウト設計書

## 概要

この文書は、訓練プリント（弱点克服トレーニング）の印刷用紙（PDF）の見た目・構成・レイアウトを定義します。

---

## 用紙仕様

| 項目 | 仕様 |
|------|------|
| 用紙サイズ | A4（210mm × 297mm） |
| 印刷面 | 両面印刷（表：問題、裏：解答・解説） |
| 余白 | 上下左右：10mm |
| フォント | Kiwi Maru（日本語）、システムフォント（数式） |

#### レイアウト仕様

| 項目 | 仕様 |
|------|------|
| 列数 | 1列（問題数が10問以下の場合）または2列（11問以上） |
| 問題番号 | (1)～(20)、太字、12pt |
| 問題式 | 数式（MathJax）、16pt（大きめ） |
| 解答欄 | 幅50mm、高さ10mm、下線 |
| 問題間スペース | 8mm（サバイバルテストより広め） |

#### レイアウト自動調整

| 問題数 | レイアウトクラス | 列数 | 解答欄余白 |
|--------|-----------------|------|-----------|
| 1～2問 | `.ultra-few` | 1列 | 上部110mm |
| 3～5問 | `.few-problems` | 1列 | 上部90px |
| 6～10問 | （デフォルト） | 1列 | 通常 |
| 11～20問 | （デフォルト） | 1列 | 通常 |

**設計意図**: 問題数が少ないレベル（Level 4～5）は、大きな余白を確保して筆算スペースとして活用

#### レイアウト仕様

|------|------|

#### 解説文の表示ルール

- **サバイバルテスト**: 解説文なし（問題数が多いため）
- **訓練プリント**: 解説文あり（各問題に詳細な指導ポイント）

---

## デザイン原則

### 1. 学習効果重視

- **大きめの文字**: 問題式16pt、解答16pt（視認性向上）
- **解説文充実**: 各問題に💡マークで指導ポイントを明記
- **お手本表示**: 解き方の手順を視覚化

### 2. 反復練習しやすさ

- **余白確保**: Level 4～5は筆算スペースとして上部110mm確保
- **タイム目標明記**: 各レベルの目標タイムを⏱️アイコンで強調
- **ワンポイントコーチング**: 💬マークで励ましメッセージ

### 3. 教師・保護者向け

- **解説文の充実**: annotationsフィールドに詳細な指導ポイント
- **色分け**: 解答は赤色で採点しやすく
- **手書き風フォント**: 解説文は親しみやすい雰囲気

## レイアウト分岐ロジック

```javascript
// 問題数に応じたレイアウトクラス選択
let layoutClass = '';
if (pattern.problems.length >= 20) {
    layoutClass = 'survival-sheet'; // 2列レイアウト
} else if (pattern.problems.length <= 2) {
    layoutClass = 'ultra-few'; // Level 5用：110mm余白
} else if (pattern.problems.length <= 5) {
    layoutClass = 'few-problems'; // Level 4用：90px余白
}
// デフォルト: 11～19問は通常の1列レイアウト
```
